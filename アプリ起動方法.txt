■2回目以降の起動手順

1.フォルダに移動
  cd chatbot

2.仮想環境を有効化 （ターミナルを閉じた場合は、再度有効化が必要です）
.\venv\Scripts\Activate.ps1

※行の先頭に 
(venv)と表示されていれば、この手順はスキップできます。

3.アプリを起動
  python app.py



■初回の起動手順
ターミナル（PowerShellやコマンドプロンプト）で以下のコマンドを順番に実行してください。

1.ディレクトリの移動 まず、アプリのフォルダ（chatbot）に移動します。
  cd chatbot

2.ライブラリのインストール 必要なライブラリをインストールします。
  pip install -r requirements.txt

3.アプリの起動 以下のコマンドでサーバーを立ち上げます。
  python app.py

4.ブラウザでアクセス 起動に成功すると、ターミナルに以下のようなメッセージが表示されます。
  Running on http://0.0.0.0:3000 ブラウザを開き、以下のアドレスにアクセスしてください。
  http://localhost:3000

注意点
APIキーの設定: 
.env
ファイルにCOHERE_API_KEYが正しく設定されているか確認してください。
設定されていない場合、アプリ起動時にエラーになります。
停止方法: ターミナルで Ctrl + C を押すとサーバーを停止できます。

起動手順
ターミナル（PowerShellなど）で、
app.py
があるフォルダ（chatbot\chatbot）に移動してから、以下のコマンドを順番に実行してください。

1既存の仮想環境を削除（macOS用のため）
powershell
Remove-Item -Recurse -Force venv
※ エクスプローラーから削除してもOKです。

2Windows用の仮想環境を作成
powershell
python -m venv venv

3仮想環境を有効化
powershell
.\venv\Scripts\activate
※ 行頭に 
(venv)
 と表示されればOKです。


4必要なライブラリをインストール
powershell
pip install -r requirements.txt


5アプリを起動
powershell
python app.py


6ブラウザでアクセス 起動したら、ブラウザで http://localhost:3000 にアクセスしてください。
もしよろしければ、これらのセットアップコマンドをこちらで実行することも可能です。その場合は「セットアップを実行して」とおっしゃってください。

Error

Thought for 2s




アプリを停止するには、以下の手順を行ってください。

ターミナル（コマンドを実行している画面）をクリックしてアクティブにします。
キーボードの Ctrl キーを押しながら C キーを押します。
これでプログラムが強制終了し、アプリが停止します。 もし停止しない場合は、もう一度 Ctrl + C を押すか、ターミナルウィンドウのゴミ箱アイコン（または「×」ボタン）を押してターミナル自体を閉じてしまっても大丈夫です。
